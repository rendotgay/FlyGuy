# FlyGuy
A Discord bot designed to be the ultimate way to count with friends!
No matter your reason to need a count, FlyGuy will make it the ultimate counting experience!

## Features
- Solo counting
- Invite a friend
- Group invites
- Leaderboards

## Invites
An invitation allows the user(s) to accept once they are ready.
Each invite features a unique GIF and rarity to create a random and engaging experience!
Once you reach a special number of invites, you get unique messages, GIFs and XP!
Not feeling an invite? You can also decline an invite!
Compete with your friends to see who can level up the fastest!
Don't have friends to invite? No worries, you can chat with FlyGuy or send him an invite, or even ask him for one!

GIF's are provided by:
- [Tenor](https://tenor.com) (Until June 30th 2026 when Tenor shuts down their API)
- [GIPHY](https://giphy.com)
- [KLIPY](https://klipy.com)

GIFs are ensured to never duplicate among friends and a list of custom search terms is specified to keep results high quality.

## Configuration
### Environment Variables
- `TOKEN` - The bot's Discord API token
- `TENOR_API` - The Tenor API key
- `GIPHY_API` - The GIPHY API key
- `KLIPY_API` - The KLIPY API key

## Usage
### Bot commands
- `/count` - Starts a solo count
- `/invite` - Invites a friend
- `/leaderboard` - Displays FlyGuy's leaderboard. Specify a user to see their statistics.
